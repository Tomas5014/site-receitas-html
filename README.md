# 🍽️ Recipe Website — Pure HTML

A recipe website built with **pure HTML**, created to practice web development fundamentals and version control.

## 📌 About the Project

A static recipe website built without any frameworks or external libraries — just HTML. The goal was to strengthen knowledge of the language's core structures and elements, while also practicing a Git and GitHub workflow.

## 🛠️ Technologies Used

- HTML5

## 📚 What Was Practiced

**HTML**
- Document structure and semantics
- Tags and elements (headings, paragraphs, sections, etc.)
- Ordered and unordered lists
- Internal and external links
- Adding and organizing images

**Version Control**
- Source control with Git
- Publishing and managing a repository on GitHub
